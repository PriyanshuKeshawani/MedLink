# MedLink Home - Healthcare Platform

A modern, premium healthcare web platform that connects patients with nearby doctors and nurses for home-based medical services.

## Features
- **Patient Dashboard**: Manage bookings, view medical history, and contact emergency services.
- **Pro Dashboard**: For doctors and nurses to accept nearby requests and track earnings.
- **Premium UI**: Glassmorphism design, smooth animations (Framer Motion), and responsive layout.
- **Service Range**: Consultations, injections, wound dressing, and monitoring.

## Tech Stack
- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Design Tokens)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## How to Run Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Open in Browser**:
   Navigate to `http://localhost:5173`

## Directory Structure
- `src/App.jsx`: Main logic and routing (Landing, Patient, Pro Dashboards).
- `src/index.css`: Design system and global styles.
- `src/main.jsx`: Entry point.
- `index.html`: Base HTML with Google Fonts.
